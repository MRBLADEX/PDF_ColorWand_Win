## PDF魔棒

适用于将白底黑字图片类型的PDF转换成护眼的色彩。

## 使用说明

1. 点击 Open 打开 PDF，加载到文件列表中，可选中按 DELETE 键删除。
2. 点击 Dist 设置输出目录。
3. 下方可调整输出背景色、文字内容颜色、转换阈值（一般不需要改动）、图像质量。
4. 点击 Transform 等待转换完成。
